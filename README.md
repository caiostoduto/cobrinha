1v1 Snake Pygame

## Summary
- [🤓 Install](#-install)
- [😎 How to run](#-how-to-run)
- [🥳 How to play](#-how-to-play)

## 🤓 Install
```bash
pip install -r requirements.txt
```

## 😎 How to run
```bash
python runner.py
```

## 🥳 How to play
- Use the WASD keys to move player0
- Use the arrow keys to move player1
